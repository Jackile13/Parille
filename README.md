### Summary
Parille is a two player strategy game based on Orthello. Each player takes turns to place tiles of their own colour, taking over the player's tiles. The player with the most tiles to their name at the end of the game wins!
Parille features Player vs AI, AI vs AI, and Player vs Player (on the same device or two devices nearby wirelessly)

Download it [here.](https://apps.apple.com/us/app/parille/id1666184682)

### Technical details
- Made purely in Swift and SwiftUI
  - SwiftUI animations for smooth and efficient animations
  - SFSymbols for familar feel and natural integration
- Basic game engine utilising the state construct to keep the game state consistent
- Achievements that trigger as soon as the player earns it with a custom, un-instrusive popup
- Statistics that track every game individually and provide a summary of all games
- Uses Apple's Multipeer Connectivity framework for local wireless multiplayer
